## Screenshot Guidelines:

1. **Resolution**: Recommended 1920x1080 for desktop, 375x812 for mobile
2. **Format**: PNG or JPG
3. **File Size**: Keep under 500KB per image (use compression tools)
4. **Content**: 
   - Use realistic data (not default lorem ipsum)
   - Show features in action
   - Include both light and dark mode for key pages
   - Demonstrate responsive design

## Tools for Screenshots:

- **Windows**: Snipping Tool, Snip & Sketch, ShareX
- **Browser DevTools**: Chrome/Firefox responsive mode
- **Online Tools**: Screely.com, Screenshot.rocks
- **Browser Extensions**: Awesome Screenshot, Full Page Screen Capture
